# 交付说明（DELIVERY）

## 版本 / 范围

- 看板三列（待办/进行中/已完成）+ 拖拽移动
- 任务 CRUD，优先级（高/中/低）与截止时间
- 搜索、防抖，截止时间与优先级筛选，优先级排序（默认高→低）
- 月历视图：仅展示有截止时间的任务，支持查看/编辑/删除/状态切换/附件操作
- 附件上传/下载：10MB 限制，MIME 白名单（image/*、application/pdf、text/plain）
- 本地持久化：SQLite `backend/data/app.db` + 附件目录 `backend/uploads/`
- 健康检查 `/health`，后端 CORS 仅允许 `http://localhost:5173`

## 回归清单

- 任务创建/编辑/删除/状态切换
- 拖拽跨列移动；失败时提示并回滚
- 月历渲染、切换月份、日期弹窗交互
- 搜索/筛选/排序（标题、截止时间、优先级）
- 附件上传、类型/大小校验、下载
- 截止时间解析与展示（含无截止时间的处理）
- 持久化：重启后任务与附件仍可访问
- 错误提示：网络异常、验证失败、上传限制、找不到任务

## 验收步骤（Axx）

- A1 空列表/错误态：初始为空时展示空状态；模拟后端停机时前端展示错误提示。
- A2 新建任务（含 datetime）：填写标题、优先级、截止时间，保存成功后列表展示。
- A3 编辑/状态切换：进入编辑修改字段；在看板或详情中切换状态并立即生效。
- A4 删除：删除任务后从列表移除，若有附件也被清理（文件与元数据）。
- A5 上传限制：尝试 >10MB 或非白名单类型，前端/后端返回限制提示，合法文件可成功上传。
- A6 下载：点击附件触发浏览器下载，文件名为原始名称。
- A7 重启持久化：停止并重启后端，已存在任务与附件仍可读取。
- A19 拖拽移动：在看板中拖拽任务跨列，状态更新并刷新成功提示。
- A20 拖拽失败回滚：模拟后端错误（停机或接口返回 500），拖拽后状态恢复原列并提示失败。
- A21 月历切换：切换月份，导航仍显示有截止时间的任务标记。
- A22 月历渲染：当月含截止时间的任务在对应日期显示标记。
- A23 月历弹窗操作：点击日期弹出列表，可编辑/删除/切换状态/管理附件。
- A24 月历 dueDate 规则：仅 dueDate 存在的任务出现；无截止时间不在月历。
- A25 月历响应式：窄屏下月历与列表布局可滚动/可用。
- A26 月历回归：切换回看板视图后数据一致。

## 已知限制 / 不做项

- 无登录、无多用户协作、无云同步；仅本地存储
- 未提供复杂日历拖拽/批量编辑；月历仅展示截止时间关联任务
- 未提供提醒/推送；需要手动查看
